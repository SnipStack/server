# Express Typescript Boilerplate
## Prerequisites
* Node.js
* Express.js
* Prisma ORM
* Eslint
* JWT
## How to use
Clone the repo
```bash
git clone https://github.com/JustFossa/express-ts-boilerplate.git
```
Install packages with yarn or npm
```bash
yarn
```
or
```bash
npm install
```
Create .env file
```env
DATABASE_URL=<your-postgres-url>
JWT_SECRET=<your-jwt-secret>
```
Run the server
```bash
yarn dev
```
or
```bash
npm run dev
```
